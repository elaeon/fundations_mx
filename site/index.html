Mexico Fundations & Trust Visualizer GitHub

This is a research tool that visualizes 10017 fundations & trust from the SAT Donatarias, covering 505,167 beneficiaries across the Mexican's states. Each rectangle's area is proportional to total beneficiaries Color shows the selected metric — toggle between patrimony, work forcs, volunteers, wages, exposure. Click any tile to view its full data. This is not a report, a paper, or a serious economic publication — it is a development tool for exploring Fundations data visually.

LLM-powered coloring: The source code includes scrapers, parsers, and a pipeline for writing custom LLM prompts to score and color fundations by any criteria. You write a prompt, the LLM scores each occupation, and the treemap colors accordingly. You could write a different prompt for any question — and re-run the pipeline to get a different coloring.
